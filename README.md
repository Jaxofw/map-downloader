# Deathrun Map Downloader

A tool used for downloading CoD4 Deathrun maps from a FastDL server.

## Installation

1. Clone the repository or download the `download_maps.py` file.
2. Install the required libraries by running the following command:
    ```
    pip install requests urllib3 beautifulsoup4
    ```
3. Move `download_maps.py` to your game's root folder or move the `usermaps` folder to the game folder afterwards.

## Usage

1. Open a terminal or command prompt.
2. Navigate to the directory where `download_maps.py` is located.
3. Run the script using the following command:
    ```
    py download_maps.py
    ```
